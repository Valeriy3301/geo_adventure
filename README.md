# Geo Data Transformer

Python-based geo data engineering application for:
- CRS transformations
- GeoJSON processing
- geometry enrichment
- ETL-style geo pipelines
- spatial analytics
- export/report generation

Technologies:
- Python
- Pandas
- Shapely
- PyProj
- Docker
- GitHub Actions

Run locally:

pip install -r requirements.txt
python app.py

Run with Docker:

docker-compose up --build
